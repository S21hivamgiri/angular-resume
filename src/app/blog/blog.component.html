Since the Object "account" is referenced instead of deep copying, it stores the memory location of the Object "account" in both the index of the Array "accounts". So when the modification of any one of the Object happens both the elements of Array "accounts" is updated as both references to same memory location. In case we need to deep copy, and expect output as  FrontEnd Buddy ,  we need to modify our code as below:
const account = {{ '{' }}
    name: 'FrontEnd',
    tag: 'Follow me on Insta',
    description: 'Help Learner to explore JS'
};

const accounts = [{{ '{' }}...account}, {{ '{' }}...account}]; // Changed here to deep copying
//as earlier it was [account, account] which shallow copies object account
accounts[1].name = 'Buddy';
console.log(accounts[0].name +' '+ accounts[1].name);
